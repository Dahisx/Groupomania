## Node version

J'ai utilise la version de node 16.13.2

## Comment lancer le projet 

Pour lancer le Frontend, veuillez tapper ces commandes ci dessous: 

```bash
cd client 
npm install
npm run dev
```

Pour lancer le Backend, Veillez tapper les commandes ci dessous : 

```bash
cd server
npm install
nodemon server
```
